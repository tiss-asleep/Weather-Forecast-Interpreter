# Weather Forecast Interpreter
A full-stack web application that transforms raw meteorological data into conversational, AI-generated insights. The system orchestrates data flow between the WeatherAPI for real-time/forecast data and Googleâ€™s Gemini 2.5 Flash-Lite for natural language processing.

Live Demo: https://weather-forecast-interpreter.onrender.com

To run the project locally:
  - Configure your environmental variables according to the examples given in .env.example file
  - Install the dependencies using the command "pip install -r requirements.txt"
  - Run the server using "python app.py"
  - Open http://127.0.0.1:5000 in your browser
